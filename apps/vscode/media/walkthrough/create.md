# Create your first diagram

Team Topologies diagrams live in `.tt` files (or `.ttm.json`). This extension opens them in a
visual editor instead of showing raw text.

**Do this now:** click **Create diagram from example** in the step on the left. (You can also open
the Command Palette with `Cmd/Ctrl+Shift+P` and run **Team Topologies: New Diagram from Example**.)

Then:

1. A save dialog appears — choose a folder and file name ending in `.tt` and confirm.
2. The file opens straight in the visual editor, pre-filled with a few teams and interactions you
   can rearrange and learn from.

Prefer a blank canvas? Run **Team Topologies: New Empty Diagram** instead. Either way, any file
ending in `.tt` or `.ttm.json` opens in this editor automatically from now on.
